# weather

Simple web app that fetches current weather for a location using the OpenWeatherMap API.

Usage
- Edit `js/script.js` and replace `YOUR_API_KEY_HERE` with your OpenWeatherMap API key.
- Open `index.html` in a browser, enter a city, and click "Get Weather".

Features
- Shows temperature, description, and weather icon.
- Background animations reflect the current weather (sunny, rain, snow, thunder, clouds, fog).
- Handles invalid locations and network/API errors with user messages.

Tech
- HTML, CSS, JavaScript (no build step required)
